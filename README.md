# Customer-Shopping-Behaviour
Customer Shopping Behaviour Dashboard built using Power BI, supported by Python-based data preprocessing and prediction generation. Includes visual insights on customer trends, purchase patterns, and category performance. Contains dataset, ML code, and PBIX file for reference.
Features

Interactive Power BI dashboard with a clean UI

Category-wise purchase and rating analysis

Age-frequency behaviour visualization

Key KPIs including total customers, average purchase value, and prediction accuracy

Python-based preprocessing, feature engineering, and model prediction

Complete dataset, notebook code, and Power BI file included

Tech Stack

Python: Pandas, NumPy, Scikit-Learn

Power BI: DAX, Power Query, Visualization

Tools: Jupyter Notebook, CSV dataset, GitHub

Project Workflow

Data loading and cleaning in Python

Feature creation and model training (Random Forest)

Export predictions into the dataset

Building interactive visuals and KPIs in Power BI

Publishing and documentation

Challenges Faced

Handling mixed data types during preprocessing

Cleaning rating values and applying custom rounding logic

Dealing with label encoding inconsistencies

Ensuring Power BI correctly read numeric columns without conversion errors

Creating accurate DAX measures for prediction accuracy

Maintaining design consistency across visuals

Python notebook / scripts for preprocessing and ML

Enhanced dataset with predictions
